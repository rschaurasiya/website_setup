# Frontend Application

### `src/components/`
- Reusable UI components
- **Examples**: `Header.jsx`, `Footer.jsx`, `BlogCard.jsx`, `Comment.jsx`, `AuthForm.jsx`

### `src/pages/`
- Full page components
- **Examples**: `Home.jsx`, `BlogDetail.jsx`, `CreateBlog.jsx`, `Profile.jsx`, `Login.jsx`

### `src/services/`
- API integration and data fetching
- **Examples**: `api.js`, `authService.js`, `blogService.js`

### `src/utils/`
- Helper functions
- **Examples**: `formatDate.js`, `validators.js`, `constants.js`

### `src/assets/`
- Static files

#### `assets/images/`
- Images, logos, icons

#### `assets/styles/`
- CSS/SCSS files
- **Examples**: `global.css`, `variables.css`, `theme.css`

### `src/config/`
- Frontend configuration
- **Examples**: `apiConfig.js`, `constants.js`

### `public/`
- Publicly accessible static files
- **Examples**: `index.html`, `favicon.ico`, `robots.txt`
